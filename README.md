## Run Locally

Clone the project

```bash
  git clone PixelOps
```

Go to the  project directory

```bash
  code PixelOps
```
Install dependencies

```bash
  npm install nodemon --save-dev
```
Open XAMP 

```bash
  Start Apache and MySQL
```

Go to phpMyAdmin http://localhost:8080/phpmyadmin/

```bash
  Click new and create a new database called pixelops
```

Start the server

```bash
  npm start
```
